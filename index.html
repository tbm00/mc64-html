MC64

A cracked, semi-anarchy Minecraft Java server with hacks, claims, gangs, ranks, an economy, and minimal moderation. We allow in-game hacks and have 'few' rules... So when you login, be careful who you trust!

Features

1.20.4 PVP & SMP: Use /togglepvp to control your pvp status!

Semi-Anarchy: Hacking, griefing, scamming, and stealing is allowed!

Server Econony: Make money with shops, auctions, warps, and jobs.

Ranking System: Unlock special commands and abilities as you play!

Player Gangs: Team up, create alliances, and toggle friendly-fire!

Multiple Worlds: We have Custom, AMPLIFIED, and Normal biomes!

Expiring Land Claims: Protect your stuff while you play, raid inactive claims once they expire!

Minimal Moderation: Staff mostly cares about minimizing exploits, dupes, and bigotry.

Open Community: Stay connected to the community via our Discord server!

Not P2W: However we do offer some minor perks for donations!

Cracked: Server is open to everyone across the globe!

History

This is the second official release of MC64! Our community started in July 2021 with the initial release of MC64. The server grew and grew, but stopped prematurely due to some critical database errors. The server went offline for a little over a year, but our community stayed alive through Discord (and also through some unofficial servers inspired by MC64). During Summer 2023, I had a bit more time on my hands, so I decide to work on getting the server up and running again. After a couple months of some trials and tribulations, I'm happy to announce the server is now officially open again!
